# paper/

- `land-restrictions-paper.html` — source template. Contains the full text
  with figure placeholders `__F1__` … `__F9__`. **Edit this file**, not the
  final one.
- `land-restrictions-paper-final.html` — assembled paper with all figures
  embedded as base64 data-URIs (self-contained, ~4 MB). Generated by
  `analysis/assemble_paper.py`; do not edit by hand. Also published as a
  Claude web artifact.
- `land-restrictions-paper.pdf` — PDF render of the final HTML.

To rebuild after changing text or figures:
`python analysis/assemble_paper.py` (figure paths are hardcoded there;
regenerate figures first via the pipeline in `analysis/README.md`).

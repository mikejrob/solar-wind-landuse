# MANIFEST — Procurement / pipeline-mortality primary sources

Compiled 2026-07-19. Companion analysis: `notes/project-pipeline-mortality.md`;
data: `data/oahu_solar_project_pipeline.csv`. These document the ECONOMIC/procurement mortality
stage that (unlike community resistance) is where Oahu solar actually dies. RFP docket = 2017-0352.

"Downloaded" = local file present; "Link-only" = retained URL (paywall / JS-gated CDMS / etc.).

- **puc-stage3-idrp.html** — PUC Stage 3 RFP + Interconnection Dispute Resolution Process page

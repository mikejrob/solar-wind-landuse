# testimony/

Curated primary testimony records for the key Chapter 205 solar-siting bills,
one subdirectory per measure (`<bill>_<year>/`). Each hearing's compiled
testimony appears as the original PDF from capitol.hawaii.gov plus a `.txt`
text extraction alongside it (same basename). Filenames encode draft,
committee, and hearing date, e.g.
`SB631_SD1_TESTIMONY_WLO-EEP_03-21-11_.PDF`.

Covered: `sb631_2011` (Act 217 — the 10%/20-acre cap), `sb2775_2014` and
`sb2658_2014` (Acts 52/55), `hb2665_2018`, `sb942_2021`, `hb969_2025`.

`sb942_2021/` is empty: the bill died without the relevant hearing record
being captured here (see `data/bills.csv` for status).

The wider raw capture (bill texts, committee reports, additional testimony)
lives in `data/raw/capitol/` (gitignored); this directory holds the curated,
tracked subset the notes cite.

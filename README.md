# Land for Solar and Wind Development on Oahu

Research compilation on land-use rules, ownership, grid proximity, and terrain
for utility-scale solar and wind siting in Hawaii (Oahu focus), with the full
legislative/documentary record behind the current rules.

- `paper/land-restrictions-paper-final.html` — the background paper (self-contained, figures embedded)
- `notes/` — per-thread research notes; every factual claim carries a primary-source citation or an UNVERIFIED flag
- `data/` — tidy, TMK-keyed CSVs (parcels x soil x ownership x transmission x slope; SUP census; campaign finance; interlocks)
- `analysis/` — reproducible Python scripts (GIS screens, figures, paper assembly)
- `testimony/` — primary testimony PDFs with text extracts
- `CLAUDE.md` — project instructions and the running verified-findings record

Large raw caches (`data/raw/`, `data/gis/` layers, `cache/`) are excluded;
`analysis/` scripts and the Wayback recipes in `notes/` regenerate them.
